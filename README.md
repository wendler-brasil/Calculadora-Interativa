Calculadora Interativa
📖 Sobre o Projeto

A Calculadora Interativa é uma aplicação desenvolvida em Java que permite realizar operações matemáticas básicas por meio de um menu interativo no terminal.

O programa oferece uma experiência simples e intuitiva, permitindo que o usuário escolha a operação desejada, informe os valores e visualize o resultado imediatamente.

Além disso, as operações de soma e subtração permitem adicionar novos números continuamente sem precisar retornar ao menu principal.
